# Library Management System

## Description
A console-based Library Management System built using Core Java and JDBC. 
The application allows managing books, members, and issue/return operations 
with MySQL database integration.

## Tech Stack
- Java (Core Java, OOP)
- JDBC
- MySQL
- Collections Framework

## Features
- Add, update, delete books
- Register library members
- Issue and return books
- View available and issued books
- Database persistence using JDBC

## Project Status
Completed

## How to Run
1. Clone the repository
2. Configure MySQL database
3. Update database credentials in the code
4. Run the main class
